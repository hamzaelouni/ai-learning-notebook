# Machine Learning — From Zero to Expert

A complete, self-paced course to take you from no ML knowledge to expert level.

## How to use this course

- Each module is a standalone markdown file with theory, examples, and exercises
- Read in order — each module builds on the previous
- Don't skip the exercises — that's where real learning happens
- Pair every module with hands-on code (Python + scikit-learn at first, then PyTorch)
- Estimated total time: **3–6 months** at 5–10 hours/week

## Prerequisites

- **Python basics** (variables, functions, loops, lists, dicts)
- **High school math** (we'll review what you need)
- That's it. Seriously.
