# Master Reference List

A curated list of resources to deepen any topic in this course. **You don't need to read all of these** — pick 1 book and 1 course as your main spine, and use the rest as reference.

---

## 📚 Books (in order of recommendation)

### Beginner — Start here

1. **Hands-On Machine Learning with Scikit-Learn, Keras & TensorFlow** — *Aurélien Géron* (3rd ed.)
   - **The single best practical ML book.** Use it as the spine of this course.
   - Each chapter has a Jupyter notebook on GitHub: https://github.com/ageron/handson-ml3

2. **Python Data Science Handbook** — *Jake VanderPlas* (free online: https://jakevdp.github.io/PythonDataScienceHandbook/)
   - Best NumPy / Pandas / Matplotlib reference.

### Intermediate

3. **An Introduction to Statistical Learning** — *James, Witten, Hastie, Tibshirani* (free PDF: https://www.statlearning.com/)
   - The most readable statistical foundation. Has Python and R versions.

4. **Pattern Recognition and Machine Learning** — *Christopher Bishop*
   - Classic, more mathematical. Read after ISL if you want depth.

### Advanced

5. **The Elements of Statistical Learning** — *Hastie, Tibshirani, Friedman* (free PDF: https://hastie.su.domains/ElemStatLearn/)
   - The "ESL" — the bible. Tough but worth it once you have ISL down.

6. **Mathematics for Machine Learning** — *Deisenroth, Faisal, Ong* (free: https://mml-book.github.io/)
   - Solid math review.

### Deep learning (after this course)

7. **Deep Learning** — *Goodfellow, Bengio, Courville* (free online: https://www.deeplearningbook.org/)

8. **Dive into Deep Learning** — *Zhang, Lipton, Li, Smola* (free: https://d2l.ai/)
   - Free, interactive, with code in PyTorch / TensorFlow / JAX.

## 📜 Foundational Papers (read after the course)

- **Linear regression / least squares** — Gauss, 1809 (~ish)
- **Perceptron** — Rosenblatt, 1958
- **Backpropagation** — Rumelhart et al., 1986
- **Random Forests** — Breiman, 2001
- **AdaBoost** — Freund & Schapire, 1997
- **XGBoost** — Chen & Guestrin, 2016
- **AlexNet** (deep learning takeoff) — Krizhevsky et al., 2012
- **Attention Is All You Need** (Transformers) — Vaswani et al., 2017
- **BERT** — Devlin et al., 2018
- **GPT-3** — Brown et al., 2020

Find them all on https://arxiv.org or https://paperswithcode.com.

## 🗺 Recommended Learning Path

If you only do 5 things after this course:

1. Read **Hands-On Machine Learning** (Géron) cover to cover
2. Watch **Andrew Ng's ML Specialization** on Coursera
3. Watch **3Blue1Brown's Linear Algebra & Calculus** playlists
4. Compete in 1 **Kaggle Getting Started** competition (e.g., Titanic, House Prices)
5. Build 1 end-to-end project from scratch and deploy it (FastAPI + cloud)

Then move to deep learning with fast.ai or the Karpathy "Zero to Hero" series.
